Wave Function Collapse

Understanding the transition from quantum possibilities to a definite outcome.

What is Wave Function Collapse?

Wave function collapse refers to the phenomenon where a quantum system transitions from a superposition of states to a single eigenstate due to measurement.

The Role of Measurement

In quantum mechanics, observing a system forces it to "choose" a definite state. For example, when measuring the position of a particle, the wave function collapses, and the particle is found in a specific location.

Schrödinger's Cat Thought Experiment

Schrödinger's cat is a famous example illustrating wave function collapse. The cat remains in a superposition of alive and dead states until observed, at which point the wave function collapses into one of the two possibilities.

Interpretations of Wave Function Collapse

  • Copenhagen Interpretation: Collapse is caused by measurement.
  • Many-Worlds Interpretation: All outcomes occur in separate universes.
  • Decoherence: Interaction with the environment causes the appearance of collapse.
